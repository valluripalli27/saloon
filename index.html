Project Overview

I designed and developed a full-featured salon and spa website with seamless navigation and a responsive user interface. The platform enables real-time service browsing, filtering, and appointment booking to ensure a smooth and intuitive user experience. Secure payments were integrated using Stripe and Apple Pay, along with features like optional tipping and a loyalty system. Additionally, I implemented location-based service filtering and customizable add-ons to personalize each booking.

Reception-First Design Approach

Designed the landing flow around first-time users making it feel like walking into a calm, welcoming lobby rather than navigating a cluttered menu.
